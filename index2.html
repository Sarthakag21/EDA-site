Exploratory Data Analysis (EDA)

From History to Tools, Techniques and Practical Applications

1. Introduction

Exploratory Data Analysis (EDA) is a crucial step in the data science process. It involves investigating datasets to summarize their main characteristics and uncover patterns, anomalies, and relationships using both statistical and visual techniques.

2. History of EDA

The concept of EDA was introduced by the American statistician John Tukey in the 1970s. Tukey emphasized the importance of visualizing data before formal modeling to better understand the data’s structure and detect underlying patterns or errors.

3. Why EDA Matters (Use Cases)

  • Understanding distribution and spread of data
  • Identifying missing data and outliers
  • Uncovering hidden patterns and trends
  • Preprocessing decisions before machine learning

Example: In healthcare, EDA is used to study patient data to detect health trends, identify risk factors, and improve decision-making.

4. Key Features of EDA

  • Descriptive statistics
  • Data visualization
  • Outlier detection
  • Correlation analysis

5. Techniques Used in EDA

5.1 Summary Statistics

Include measures like mean, median, mode, standard deviation, skewness, and kurtosis to understand the basic distribution of the data.

5.2 Bar Chart

A bar chart is used to represent categorical data with rectangular bars. The length of each bar is proportional to the value it represents.

import matplotlib.pyplot as plt
categories = ['A', 'B', 'C']
values = [10, 20, 15]
plt.bar(categories, values)
plt.title('Bar Chart Example')
plt.show()

5.3 Histogram

Histograms show the frequency distribution of a numerical variable. It's useful to understand the shape of the data distribution.

plt.hist(df['age'], bins=10)
plt.title('Age Distribution')
plt.show()

5.4 Box Plot

A box plot shows the median, quartiles, and outliers of a dataset. It’s useful for identifying outliers and understanding distribution.

import seaborn as sns
sns.boxplot(x=df['salary'])

5.5 Scatter Plot

Scatter plots are used to study the relationship between two continuous variables.

sns.scatterplot(x='height', y='weight', data=df)

5.6 Correlation Matrix

A correlation matrix shows how variables are related to each other, with values ranging from -1 to 1.

import seaborn as sns
sns.heatmap(df.corr(), annot=True, cmap='coolwarm')

6. Tools Commonly Used

  • Python: pandas, matplotlib, seaborn, plotly
  • R: ggplot2, tidyverse, dplyr
  • Excel: Pivot tables, charts, summary stats
  • Power BI / Tableau: Drag-and-drop dashboards and interactive visualizations

7. Real-World EDA Example: Titanic Dataset

The Titanic dataset is a classic for practicing EDA. It contains information about passengers on the Titanic, including their age, class, gender, and survival status.

import seaborn as sns
import pandas as pd
import matplotlib.pyplot as plt

df = sns.load_dataset('titanic')
print(df.head())
sns.countplot(x='survived', data=df)
plt.title('Survival Count')
plt.show()

This basic plot shows how many people survived vs. died on the Titanic.

8. Conclusion

EDA is the foundation of any successful data project. It ensures that data scientists deeply understand their data before building predictive models. Visualizations and statistical summaries make patterns and problems in the data transparent and easy to interpret.
